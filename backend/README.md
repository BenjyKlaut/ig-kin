Backend (Express + SQLite prototype)

Requirements: Node.js >= 16

Install

```powershell
cd backend
npm install
```

Start

```powershell
npm run start
```

The server creates a `data.db` file in `backend/` on first run and seeds a demo user:

- matricule: `AGENT001`
- password: `password123`

This is a prototype for local testing only.
